# Reinforcement Learning

From-scratch **NumPy implementations** of classical reinforcement learning algorithms following *Sutton & Barto* Parts I–II — tabular methods, linear function approximation, and policy gradients.

Theoretical foundations live in [`Sequential-Decision-Making`](https://github.com/SaiSampathKedari/Sequential-Decision-Making). Deep-RL methods live in [`Deep-Reinforcement-Learning`](https://github.com/SaiSampathKedari/Deep-Reinforcement-Learning).

## Status
Early stage. Current contents:
- `reports/` — LaTeX derivations for Monte Carlo control (with and without exploring starts) and eligibility traces.

Implementations and notebooks coming soon.
